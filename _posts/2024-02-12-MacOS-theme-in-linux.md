## 1. Theme

![Theme](../assets/img/MacOS_Post/Theme.png){: .mx-auto.d-block :}

### Install the theme

Run 
```
git clone https://github.com/vinceliuice/WhiteSur-gtk-theme.git --depth=1 
cd WhiteSur-gtk-theme
sudo ./install.sh
```

This install the default whitesur theme in our machine
